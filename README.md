## 📖 Sobre o Projeto

A **Floricultura Web** é um e-commerce de flores e buquês desenvolvido para oferecer uma experiência de compra agradável e intuitiva. O site permite que os clientes naveguem pelos produtos, adicionem ao carrinho e finalizem suas compras de forma segura.

### 🎯 Objetivo

Criar uma plataforma online completa para venda de flores, com:
- Interface atraente e responsiva
- Sistema de autenticação de usuários
- Carrinho de compras funcional
- Integração com Firebase para persistência de dados

## ✨ Funcionalidades

- ✅ **Autenticação de usuários** com Google (Firebase Auth)
- ✅ **Catálogo de produtos** (Buquês, Cestas )
- ✅ **Carrinho de compras** com localStorage
- ✅ **Formulário de contato** com salvamento no Firestore
- ✅ **Design responsivo** (Desktop, Tablet e Mobile)
- ✅ **Menu dropdown** para categorias de produtos
